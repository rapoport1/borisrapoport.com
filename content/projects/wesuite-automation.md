## WeSuite Automation Framework

Designed and implemented a comprehensive **end-to-end testing infrastructure** for WeSuite's document creation and automation platform. The framework enables rapid development cycles with confidence through continuous integration.

### Key Features

- **Headless browser testing** with Cypress for reliable, maintainable E2E tests
- **Containerized environments** using Docker for consistency across development and CI/CD
- **GitHub Actions pipeline** for automatic test runs on every commit
- **Test parallelization** reducing feedback loop from 20+ minutes to <5 minutes
- **Visual regression testing** for UI consistency across releases

### Impact

Reduced regression testing time by **80%**, caught critical bugs before production, and enabled the team to iterate faster without sacrificing stability. Established testing best practices that became the foundation for the broader QA program.

**Status:** Active and evolving with the product.
