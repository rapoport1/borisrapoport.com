## Planetary Nebulae Spectral Analysis

An undergraduate physics research project analyzing **emission line spectra** from planetary nebulae to extract physical properties and elemental composition.

### Research Objectives

- Measure emission line wavelengths and intensities from Hubble Space Telescope spectroscopic data
- Calculate electron temperature and density from forbidden line ratios
- Determine chemical abundance patterns relative to solar values
- Map velocity fields across nebular structures

### Methodology

- Image processing and calibration using Python and NumPy
- Spectral line fitting using Gaussian deconvolution
- Physical modeling based on photoionization equilibrium
- Statistical analysis of measurement uncertainties
