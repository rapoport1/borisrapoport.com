## MTG Scorekeeper

A lightweight, responsive web application for keeping track of **Magic: The Gathering** game state. Perfect for casual games, Friday Night Magic, or tournament tracking.

### Features

- **Real-time score tracking** with easy increment/decrement controls
- **Poison counter support** for Infect mechanic
- **Mobile-optimized interface** for comfortable one-handed play
- **Smooth animations** and clean glass morphism design
- **No external dependencies** — pure HTML/CSS/JavaScript for blazing-fast load times

### Design Philosophy

Built with the same design principles as the rest of my portfolio—glass morphism, cosmic colors, minimal dependencies. The interface stays out of the way so you can focus on the game.
